pytaco
======


Stop ordering Pizza for your meetup! Order Taco!

Inspired by `pizza.py <https://github.com/BostonPython/fingertools/blob/master/pizza.py>`_.

Installation
============

Requires Python 3.6

::

   pip3 install pytaco

Once installed, on the command line, type::

   pytaco

and press Enter.
